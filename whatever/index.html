ابزارها+هرچی

تمام چیزهایی که در دسته‌بندی‌های دیگر قرار نمی‌گیرند در «هرچی» گذاشته می‌شوند، یا آن‌طوری که یک هکر راکو آن را می‌نویسد، *.

ویرایشگر کد، محیط برنامه‌نویسی یک‌پارچه

  • CommaIDE: طراحی شده به ویژه برای راکو. ویرایش کامیونیتی به صورت رایگان در دسترس است، ویژگی‌ها و به‌روزرسانی‌های بیشتر در برنامه‌ی حامی مالی.
  • Atom:
    • Raku syntax highlighter، پلاگینی برای برجسته‌سازی نحو راکو. دستورالعمل‌های readme را دنبال کنید.
    • Raku script runner
    • تنظیم پلاگین‌های بالا و دیگر و تبدیل اتم به یک محیط یک‌پارچه برنامه‌نویسی.
    • Raku Editor Tools، پلاگینی برای اتم که مجموعه‌ای از ابزارهای ویرایش راکو را فراهم می‌کند. (ممکن است با آخرین اتم سازگار نباشد)
  • Vim: Raku syntax highlighting برجسته‌سازی نحو راکو در ویم.
  • Neovim: vim-perl6 plugin
  • Emacs:
    • raku-mode، یک major-mode ایمکس برای راکو که برجسته‌سازی نحو و کارهای بیشتری انجام می‌دهد.
    • flycheck-raku، بررسی نحو راکو با Flycheck.
    • Unified Raku + Perl + Moose/MooseX::Declare برجسته‌ساز نحو
    • Spacemacs: پوشش ایمکس با کلیدهای ویم و چیزهای بیشتر
    • Doom Emacs: پوشش ایمکس با کلیدهای ویم و چیزهای بیشتر
  • Nano: Raku syntax highlighting
  • Padre, the Perl IDE: Padre::Plugin::Perl6، برجسته‌ساز نحو و اجرای کد راکو
  • CudaText: لکسر راکو در مدیر افزونه موجود است.

متفرقه

  • Inline::Perl6, تعبیه کردن کد راکو در پرل
